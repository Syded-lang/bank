## OPTION 1: Generate New Personal Access Token (PAT)

### Step 1: Generate Token
1. Go to: https://github.com/settings/tokens
2. Click: "Generate new token" → "Generate new token (classic)"
3. Give it a name: "Bank Project Deploy"
4. Select scopes:
   ✅ repo (all)
   ✅ workflow (if using GitHub Actions)
5. Set expiration: Choose "No expiration" or custom
6. Click: "Generate token"
7. **COPY THE TOKEN IMMEDIATELY** (you won't see it again!)

### Step 2: Use Token to Push
After generating the token, run these commands:

```bash
cd "/Users/m2/Desktop/Files-1 3"

# Set up credential helper
git config credential.helper store

# Add remote with your username and new token
git remote set-url origin https://Syded-lang:YOUR_NEW_TOKEN_HERE@github.com/Syded-lang/bank.git

# Push to GitHub
git push -u origin main
```

Replace `YOUR_NEW_TOKEN_HERE` with your actual token.


## OPTION 2: Use SSH Key (Recommended for Future)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### Step 1: Generate SSH Key
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
# Press Enter to accept default location
# Press Enter twice for no passphrase (or set one)
```

### Step 2: Copy Public Key
```bash
cat ~/.ssh/id_ed25519.pub
# Copy the entire output
```

### Step 3: Add to GitHub
1. Go to: https://github.com/settings/keys
2. Click: "New SSH key"
3. Title: "MacBook" (or any name)
4. Key type: Authentication Key
5. Paste your public key
6. Click: "Add SSH key"

### Step 4: Test and Push
```bash
cd "/Users/m2/Desktop/Files-1 3"

# Change remote to SSH
git remote set-url origin git@github.com:Syded-lang/bank.git

# Test connection
ssh -T git@github.com

# Push
git push -u origin main
```

## TROUBLESHOOTING
━━━━━━━━━━━━━━━━━

### Error: "Permission denied"
- Token is invalid/expired → Generate new one
- Token lacks 'repo' scope → Regenerate with correct scopes

### Error: "Authentication failed"
- Wrong username in URL → Should be your GitHub username
- Token expired → Generate new token

### Error: "SSH key not found"
- SSH key not generated → Follow SSH setup steps
- SSH key not added to GitHub → Add it via GitHub settings
