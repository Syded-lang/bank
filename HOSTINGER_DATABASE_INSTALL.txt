### Error: "No such file or directory"

**Solution:** Find where your files are:

```bash
# Find database.sql location
find ~/public_html -name "database.sql" -type f

# This will show the exact path
```

### Error: "Access denied for user"

**Solution:** Check your database credentials:

```bash
# Test database connection
mysql -u u299375718_user -p'Fishpoder123' u299375718_db -e "SELECT 1;"

# If this works, credentials are correct
```

### Error: "Deprecated program name"

This is just a warning (MariaDB vs MySQL). Use this instead:

```bash
mariadb -u u299375718_user -p'Fishpoder123' u299375718_db < database.sql
```

### Database file is too large

If upload fails, try:

```bash
# Split the import (from SSH)
cd ~/public_html/install

# Import in chunks
head -n 5000 database.sql | mysql -u u299375718_user -p'Fishpoder123' u299375718_db
tail -n +5001 database.sql | mysql -u u299375718_user -p'Fishpoder123' u299375718_db
```
