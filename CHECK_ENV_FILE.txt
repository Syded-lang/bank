## IF .ENV FILE EXISTS BUT WRONG LOCATION

The installer has a BUG at line 333 - it tries to create .env at wrong path!

Look at this line in the installer:
```php
$envpath = dirname(__DIR__, 1) . '/core/.env';
```

This is WRONG! When running from `core/public/install/`, this creates:
`core/public/core/.env` (WRONG!)

Should be:
```php
$envpath = dirname(__DIR__, 2) . '/.env';
```

This would create:
`core/.env` (CORRECT!)
